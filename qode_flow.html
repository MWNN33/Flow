How QODE Flow works

  • /zoom, /meet, /teams vytvoří schůzku a otevřou panel s pozvánkou — jedním klikem se odešle e-mailem všem účastníkům.
  • AI asistent rozumí přirozenému jazyku — řekni mu „Naplánuj Zoom zítra ve 14:00 s Jardou" a udělá to sám.
  • Hlas → text funguje v AI asistentovi i v hlasových poznámkách přes rozpoznávání řeči prohlížeče (nejlépe Chrome/Edge).
  • Focus plán rozdělí tvůj den na bloky, takže tě nic nevytrhne ze soustředění.
  • Naplánovaný e-mail otevře tvůj mailový klient předvyplněný ve zvolený čas.

Command reference

/zoom Team sync 2026-06-25 14:00 alice@co.com bob@co.com
→ Creates Zoom meeting, queues invites, auto-joins if time is now.
/meet 1:1 with Sam 15:30 sam@co.com
→ Same as /zoom, using Google Meet.
/teams Sprint review 16:00 dev@co.com
→ Same as /zoom, using Microsoft Teams.
/event Dentist 2026-06-26 09:00 low
→ Opens a new calendar event pre-filled.
/note Idea for the launch email
→ Opens a voice note with that headline, ready to record.
/email to:alice@co.com subject:"Friday update" at:2026-06-25 09:00
→ Opens a draft pre-filled with the details you included.
/plan [optional date like 2026-06-25]
→ Opens Focus plan for that day.
